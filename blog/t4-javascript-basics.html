Explain the difference between accessing data from arrays and object literals

When accessing data from arrays you need to use bracket notation to specify the exact bit (or bits) of data you want to access e.g. for an array called 'fruitSalad' you'd access multiple bit of data from it using numbers like this fruitSalad[0,4]. This is because the position of data in arrays is fixed and have a 0 based index, meaning the bits of data are numbered from 0,1,2,3 and so on. Whereas accessing data from an object literal requires you to specify the data using dot notation e.g. fruitSalad.mango or, bracket notation e.g. fruitSalad["mango"].

Explain what functions are and why they are useful

Functions are blocks of code which can be used repeatedly to perform a particular task. To put it simply, a function takes in inputs, does something to them and produces an output, e.g. taking and input of a name from the user and returning a full sentence including the user's name.
