What is scope in javascript? I will explain scope in the context of functions; When a variable is declared outside of a function e.g. var food = "chips" function cooking(){} that variable has GLOBAL scope. This means that the variable's value ("chips") is true both outside and inside of any function. You can change the value of that variable inside of a function by re-declaring it inside the functiom, but if you use it again outside of the function, its value will remain that same as when it was first declared. A variable with LOCAL scope is a varible whose value is true only inside of a function. When you only declare a variable inside a function e.g. function cooking() { var food = "cake" return food } that variable will only hold that value ("cake") inside of that function.
